# pfp-data-structures
Prefix-Free Parsing data structures

### Download

```console
git clone https://github.com/maxrossi91/pfp-data-structures
```

### Compile

```console
mkdir build
cd build; cmake ..
make
```

#### Debug

```console
mkdir build
cd build; cmake -DCMAKE_BUILD_TYPE=Debug ..
make
```

### Run

```console
./test/src/pfp_ds_test ../data/yeast.fasta
```
